RUSC – Roskilde University Shear Code

Publication of an open source version of the RUSC analysis code for determining frequency-dependent shear modulus using the piezoelectric shear rheometry technique (PSG devices). The technique achieves unprecedented frequency range, typically 1 mHz to 50 kHz. With his publication we present a clean and structured version of the research code, allowing for implementation at other labs.

Bo Jakobsen, Tina Hecksher, and Tage Christensen, RUSC (Roskilde University Shear Code), Zenodo (2024). doi:10.5281/zenodo.11029948
